# New Product Introduction (NPI) Guide

NPI 阶段是产品从概念到量产的关键过程。作为工艺工程师，我们在每个阶段都有特定的交付物。

## 阶段概览

1.  **EVT (Engineering Validation Test)**
    *   验证设计可行性
    *   识别主要工艺风险
    *   DFM (Design for Manufacturing) 报告

2.  **DVT (Design Validation Test)**
    *   验证设计成熟度
    *   锁定工艺参数 (DOE)
    *   模具/治具验证

3.  **PVT (Production Validation Test)**
    *   验证量产能力
    *   良率爬坡 (Yield Ramp)
    *   CT (Cycle Time) 优化

4.  **MP (Mass Production)**
    *   持续改进
    *   异常处理
