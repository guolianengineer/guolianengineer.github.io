# R语言学习路径：
一、R语言基础入门
1. [Hands-On Programming with R — Garrett Grolemund](https://rstudio-education.github.io/hopr/)

2. [R for Data Science (2e) — Hadley Wickham](https://r4ds.hadley.nz/)<br>
We’ve made a few assumptions about what you already know to get the most out of this book. You should be generally numerically literate, and it’s helpful if you have some basic programming experience already. If you’ve never programmed before, you might find Hands on Programming with R by Garrett to be a valuable adjunct to this book.


3. [R Cookbook (2nd Edition)](https://rc2e.com/)


二、数据可视化（ggplot2）系统学习
1. [R Graphics Cookbook (2nd Edition)](https://r-graphics.org/)

三、统计学基础
1. Practical Statistics for Data Scientists

四、制造业相关主题（DOE、SPC、MSA）
1. R Companion to Design and Analysis of Experiments — Max Kuhn
2. Introduction to Design and Analysis of Experiments with R

五、SPC、质量管理（如 Minitab 替代功能）
1. Applied Statistics for Quality and Productivity Using R
内容包含：<br>
	•	控制图<br>
	•	过程能力分析<br>
	•	假设检验<br>
	•	可靠性分析<br>
	•	pqcc、qcc 的应用<br>
非常适合希望从 Minitab 转向 R 的用户。<br>
2. R package qualityTools documentation
qualityTools 专门支持：<br>
	•	R&R<br>
	•	Taguchi<br>
	•	DOE<br>
	•	控制图<br>
可从包自带的 vignette 学习。<br>

六、完整学习路径<br>
我给你一个 半年内可掌握 R + 数据分析 + DOE 的路线图：<br>

1. 入门阶段（2–4 周）<br>
	•	R for Data Science (2e) 前 12 章<br>
	•	学会 tidyverse、ggplot2 基本操作<br>

2. 数据可视化（2–3 周）<br>
	•	《R Graphics Cookbook》<br>
	•	熟悉直方图、箱线图、分布图、趋势图、分组可视化<br>

3. 统计基础（2–4 周）<br>
	•	《Practical Statistics for Data Scientists》<br>
	•	配合 StatQuest 统计视频<br>

4. 质量分析（4–6 周）<br>
	•	SPC、能力分析：学习 qcc<br>
	•	MSA：学习 qualityTools<br>
	•	工厂常用统计方法<br>

5. DOE（8–12 周）<br>
	•	“R Companion to Design and Analysis of Experiments”<br>
	•	FrF2、rsm 包<br>
	•	设计、分析、优化制造工艺<br>

这条路线能让你从“只会用软件”变成“真正理解分析方法的人”。<br>
