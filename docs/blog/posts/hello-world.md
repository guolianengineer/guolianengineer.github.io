# 网站全新改版：从 Jekyll 到 MkDocs Material

大家好，欢迎来到改版后的“果链工程师”知识库！

为了提供更好的阅读体验和更专业的文档结构，我们将网站从原来的 Jekyll 主题迁移到了 **Material for MkDocs**。这次改版带来了以下新特性：

*   **更清晰的导航**：多级折叠侧边栏，轻松浏览《制造技术》的庞大章节。
*   **深色模式**：点击右上角切换，夜间阅读更护眼。
*   **即时搜索**：无需跳转，秒级查找任何技术名词。
*   **博客专区**：我们将在这里分享最新的行业动态和工程心得。

希望大家喜欢这个新家！

<!-- more -->

## 为什么改版？

随着《制造技术》部分的内容不断扩充（目前已涵盖从材料、铸造到微纳制造的完整体系），原来的单页侧边栏已经难以承载如此复杂的层级结构。MkDocs Material 的多级导航完美解决了这个问题。
